# Bright Smile Dental Clinic Website

A modern and clean dental clinic website built with **HTML5** and **CSS3**.  
Features a responsive layout with services, pricing, featured treatments, and an image gallery, designed with a professional UI.

## Technologies Used
- HTML5
- CSS3

## Project Structure
Bright-Smile-Dental-Clinic/
│
├─ dentist.html
├─ dentist.css
├─ favicon.ico
├─ images/
│ ├─ header-img.jpg
│ ├─ img1.jpg
│ ├─ img2.jpg
│ └─ ... (all gallery images)

## Features
- Responsive navigation bar
- Hero header with pricing highlights
- Services section with cards
- Featured treatment section
- Instant Grams image gallery
- Professional and clean design

## How to Use
1. Clone or download the repository
2. Open `dentist.html` in a browser
3. Enjoy exploring the website
